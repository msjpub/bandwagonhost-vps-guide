## 搬瓦工打不开，还有什么办法？

试过 bwh81.net 还是连不上的情况比较少见，一般可以这样处理：

1. **换 DNS**：把 DNS 改成 8.8.8.8 或者 1.1.1.1，有时候运营商默认 DNS 对某些域名解析有问题
2. **清除浏览器缓存**：旧的 DNS 缓存偶尔会导致重定向错误
3. **换网络环境**：比如切到手机热点试一下，有时候固定宽带的 ISP 层级会有屏蔽
4. **换到其他镜像**：bwh89.net、bwh88.net 依次尝试

一般来说，bwh81.net 访问不了是临时问题，过一会儿自己就好了。如果持续超过一天，可以提工单联系官方客服，客服会用 @bandwagonhost.com 后缀的邮件回复，可以借此验证是否为官方渠道。
